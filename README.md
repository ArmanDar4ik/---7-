## Описание проекта
Современная адаптивная страница в стиле киберпанк для медицинской платформы "Южный Мед-Облако"

## Структура проекта

page/
├── index.html # Основной HTML файл
├── components/ # HTML компоненты
├── assets/
│ ├── css/ # Стили
│ └── js/ # JavaScript модули
└── images/ # Изображени

## Функциональность
- Анимации с использованием GSAP
- Адаптивный дизайн
- Форма обратной связи
- Парящие частицы
- Киберпанк сетка

## Технологии
- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- ES6 Modules
- GSAP для анимаций
