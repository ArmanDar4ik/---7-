# Лабораторная работа 7

## Введение
Разработка современного landing page в стиле киберпанк для медицинской платформы.

## Архитектура проекта
Проект разделен на модули:
- **HTML компоненты** - разбивка интерфейса на переиспользуемые части
- **CSS модули** - разделение стилей на логические блоки
- **JavaScript модули** - ES6 модули для разных функциональностей

## Библиотеки и их безопасность

### Подключенные библиотеки:

1. **GSAP (GreenSock Animation Platform) v3.12.2**
   - Плагин: ScrollTrigger
